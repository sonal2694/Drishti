# Python Scripts

- ACLED_K_Most_Active_Cities notebook gives top K cities
- NewsScraper scrapes news for required dates and locations
- NewsTextCleaner combines text only fields for all articles on a particular date and location. It also cleans the data.
- NewsIntelligence notebook builds the Final dataset and runs ML algorithms on it

# FastText

- Go inside fasttext directory
- Train and Test datasets will be created by NewsIntelligence jupyter notebook
- Train model: ./fastText-0.2.0/fasttext supervised -input ./train_set.txt -output ./latest_model -label __label__
- Test model: ./fastText-0.2.0/fasttext test ./latest_model.bin ./test_set.txt
