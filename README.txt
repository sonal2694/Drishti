# Python Scripts

- ACLED_K_Most_Active_Cities notebook gives top K cities in India, Thailand and Indonesia where k = 15 , 6 and 6 respectively.
- NewsScraper scrapes news for required dates and locations
- NewsTextCleaner combines title fields for all articles on a particular date and location. It also cleans the data.
- NewsSentiment finds the sentiment of the news collected on a particular date and location.
- NewsIntelligence notebook builds the Final dataset and runs ML algorithms on it.

Folder "data" contains a sample file for reference on how the data is stored.

To run the application:
- Run the app.py file stored in scripts folder using the command "python3 app.py"
- Go to the folder drishti-front-end on a terminal and run the command "npm install" to install the node_modules.
- Execute the command "npm start" to run the front-end app.
- Go to localhost:3000 to use the application.


Prerequisites:
- node
- python 3
